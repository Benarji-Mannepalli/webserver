L2 L3 Protocol Development

OSI/TCP-IP Fundamentals

IP Communication Fundamentals :

This module is designed for getting expertise in IP Communications.
1. OSI Introduction
2. DataLink: Ethernet, PPP, ATM, FR, 802.11, ARP,RARP
3. NetworK: IPv4, IPv6, ICMP
4. Transport: TCP, UDP, SCTP
5. Session: RTP,RTCP
6. Presentation: SSLv3
7. Application: HTTP, HTTPS, FTP, SFTP, TFTP, SSH, Telnet, DNS, DHCP
8. Tools : Wireshark, GNS3, Cisco Packet Tracer

L3 Protocols

Layer 3 Protocols :

This module's goal is to get expertise on Layer 3 Protocols
1. Static Routing
2. Dynamic Routing Protocols Introduction
3. RIPv2 & RIPv3 : Routing Information Protocol
4. OSPFv2 & OSPFv3 : Open shortest Path First
5. EIGRP: Enhanced Interior Gateway Protocol
6. BGPv4: Border Gateway Protocol
7. IGMP: Internet Group Management Protocol
8. MPLS: Multi-Protocol Label Switching
9. GRE: Generic Routing Encapsulation protocol
10.Tools: Wireshark, GNS3

L2 L3 Protocol Development

Protocol Development Practicals :

This module is designed for getting expertise in Protocol Development
1. Protocol Development Introduction
2. Protocol State Machine Development
3. Protocol Logic Implementation
4. Protocol Coder/Decoder (CoDec) Development
5. Protocol Socket Programming /Trasport Module Development
6. Code Management using SubVersioN (SVN)
